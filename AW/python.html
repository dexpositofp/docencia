Python Programming

Learn the fundamentals of programming using Python — a simple, powerful, and widely-used language in education and industry.

1. Introduction to Python

Python is a high-level, general-purpose programming language known for its readability, simplicity, and wide range of applications. It was created by Guido van Rossum and first released in 1991. Today, Python is one of the most popular languages in education, science, web development, data analysis, artificial intelligence, and more.

One of Python's biggest strengths is its clear and straightforward syntax, making it an ideal first language for beginners. Unlike other programming languages that require complex structures to get started, Python allows you to write useful programs with just a few lines of code.

1.1 Why Learn Python?

  • Easy to learn: Python has a clean, English-like syntax that reduces the learning curve.
  • Versatile: Use Python for web development, games, automation, data science, machine learning, etc.
  • In-demand: Python is consistently among the top programming languages in job markets worldwide.
  • Huge community: Thousands of libraries, tutorials, and tools are available for every need.

1.2 What Can You Build with Python?

  • Web applications (e.g., using Flask or Django)
  • Automated scripts and bots
  • Games and visual applications (e.g., with Pygame)
  • Data analysis and visualizations
  • Artificial Intelligence and Machine Learning models
  • IoT and robotics projects (e.g., Raspberry Pi)

Did you know? Companies like Google, Netflix, Instagram, and Spotify all use Python in their tech stacks.

2. Basic Syntax and First Programs

Before diving into complex programming structures, it's essential to understand the basics of how Python code is written and executed. Python is designed to be beginner-friendly, readable, and consistent. In this section, you'll learn how to display messages, store data in variables, receive input from the user, and add comments to explain your code.

2.1 Printing Output

One of the first things we learn in any language is how to make the computer "talk" to us. In Python, we use the print() function to display text or values on the screen. 

print("Hello, world!")

You can also print numbers, variables, or results of calculations: 

print(42)
    print("2 + 3 =", 2 + 3)
    
    name = "Maria"
    print("Welcome", name)

Notice how you don't need to add any semicolons (;) at the end of each line. Python uses new lines to separate instructions.

2.2 Variables and Data Types

Variables are like containers that hold information for your program to use. You can store numbers, text, or other data types inside them. Python automatically detects the type of data you assign: 

name = "Lucas"         # string (text)
    age = 20               # integer (whole number)
    height = 1.75          # float (decimal number)
    is_student = True      # boolean (True or False)

If you're unsure what type a variable is, use the type() function: 

print(type(name))     # <class 'str'>
    print(type(age))      # <class 'int'>
    print(type(height))   # <class 'float'>
    print(type(is_student)) # <class 'bool'>

Important Notes:

  • Variable names should be descriptive: use user_age instead of x.
  • They must start with a letter or underscore, and they are case-sensitive.
  • You don't need to declare the type — Python figures it out for you.

2.3 Comments

Comments are lines of text that Python ignores when running the code. They're used to explain what the code does, making it easier for others (or yourself) to understand it later.

  • Single-line comments start with #
  • Multi-line comments are usually written using triple quotes """, although this is optional
# This line explains what the code does
    user_name = "Ana"  # Store the user's name
    
    """
    This is a longer comment
    that spans multiple lines.
    Useful for documentation.
    """

2.4 Input from the User

You can ask the user to enter information while the program runs by using the input() function. The value returned is always a string: 

name = input("What's your name? ")
    print("Hello,", name)

If you want to get a number from the user, you must convert the input using int() or float(): 

age = int(input("Enter your age: "))
    print("Next year you will be", age + 1)

Warning: If you try to perform calculations without converting the input, you’ll get an error. Python doesn’t automatically convert text to numbers.

2.5 Indentation and Structure

Python uses indentation (spaces at the beginning of a line) to define blocks of code. For example, inside an if condition or a loop, all code must be indented equally. 

age = 18
    
    if age >= 18:
        print("You can vote")
        print("Welcome to adulthood!")
    print("This line is outside the if block")

You’ll usually indent with 4 spaces. Do not mix tabs and spaces, or you'll get an error!

3. Control Flow: Conditions and Loops

Programming is not just about executing instructions line by line — it's about making decisions and repeating actions when necessary. In Python, we use control flow structures like if statements and loops (while and for) to control how our program behaves depending on different situations.

3.1 Conditional Statements: if, elif, else

With conditionals, we can tell the program to execute code only when a specific condition is met. 

age = int(input("Enter your age: "))
    
    if age >= 18:
        print("You're an adult.")
    elif age >= 13:
        print("You're a teenager.")
    else:
        print("You're a child.")
  • if checks a condition — if it’s true, the block runs.
  • elif means “else if” — it checks a second condition if the first is false.
  • else captures all other cases.

3.2 Comparison Operators

To evaluate conditions, we use comparison operators:

  • == equal
  • != not equal
  • > greater than
  • < less than
  • >= greater than or equal to
  • <= less than or equal to
score = 85
    
    if score == 100:
        print("Perfect!")
    if score != 0:
        print("Not zero!")

3.3 Logical Operators

Sometimes we need to check multiple conditions using logical operators:

  • and → both conditions must be true
  • or → at least one condition must be true
  • not → inverts the condition
age = 25
    has_id = True
    
    if age >= 18 and has_id:
        print("Access granted")

3.4 while Loop

A while loop repeats as long as the condition is True. 

counter = 1
    
    while counter <= 5:
        print("Counter:", counter)
        counter += 1

Be careful with infinite loops! Make sure your condition eventually becomes False.

3.5 for Loop

A for loop repeats a block a specific number of times. We usually use it with range(). 

for i in range(1, 6):
        print("Iteration:", i)

You can also loop through a list: 

fruits = ["apple", "banana", "cherry"]
    
    for fruit in fruits:
        print("I like", fruit)

3.6 break and continue

Use break to exit a loop early, and continue to skip to the next iteration. 

for i in range(1, 10):
        if i == 5:
            break   # stops loop at 5
        print(i)
for i in range(1, 6):
        if i == 3:
            continue  # skip 3
        print(i)

3.7 Nested Conditions and Loops

You can nest if statements or loops inside others, but keep indentation clear and organized. 

for number in range(1, 6):
        if number % 2 == 0:
            print(number, "is even")
        else:
            print(number, "is odd")

4. Functions and Code Reuse

As programs grow in size and complexity, repeating code becomes inefficient and error-prone. This is where functions come into play. Functions allow you to group blocks of code under a name so they can be reused easily throughout your program. Learning to use functions effectively is a key step in becoming a confident and efficient programmer.

4.1 What Is a Function?

A function is a reusable block of code designed to perform a specific task. You define a function once, and you can “call” it (execute it) as many times as needed, even with different data.

To define a function in Python, use the def keyword followed by a name, parentheses (), and a colon :. Inside the parentheses, you can specify input values (called parameters). The indented block that follows is the body of the function. 

def greet(name):
        print("Hello, " + name + "!")
    
    greet("Alice")
    greet("Bob")

In this example, greet is a function that takes one parameter called name. When called, it prints a personalized greeting.

4.2 Return Values

Often, functions do more than display information — they compute results. You can use the return keyword to send data back to the part of the program that called the function. 

def add(a, b):
        return a + b
    
    result = add(3, 5)
    print(result)  # Output: 8

The function add() calculates the sum of two numbers and returns the result. This makes it more versatile than simply printing the value inside the function.

4.3 Parameters vs Arguments

  • Parameters are the placeholders defined in the function header.
  • Arguments are the actual values you pass when calling the function.

Example:

def multiply(x, y):   # x and y are parameters
        return x * y
    
    multiply(4, 3)         # 4 and 3 are arguments

4.4 Function Scope

A variable’s scope defines where it can be accessed. Variables defined inside a function are local and exist only during the function's execution. This helps prevent accidental changes to variables from other parts of the code. 

def show():
        message = "Hi!"
        print(message)
    
    show()
    print(message)  # Error: 'message' is not defined outside the function

4.5 Default Parameters

You can assign default values to parameters. These are used when no argument is provided for that parameter. 

def greet(name="friend"):
        print("Hello, " + name + "!")
    
    greet()           # Uses default: friend
    greet("Carlos")   # Uses argument: Carlos

4.6 Keyword Arguments

Functions with multiple parameters can be called using keyword arguments (naming the parameter), which improves readability and avoids mistakes. 

def student_info(name, age, grade):
        print(f"{name} is {age} years old and in grade {grade}.")
    
    student_info(age=15, name="Eva", grade=9)

4.7 Why Use Functions?

  • They help avoid repeating code.
  • They make programs easier to understand and maintain.
  • They allow collaboration by dividing tasks into smaller parts.
  • They enable modular programming: writing code in organized, logical pieces.

4.8 Common Mistakes to Avoid

  • Forgetting to return a value when needed.
  • Using variables outside their scope.
  • Confusing parameter names with arguments.
  • Calling a function before defining it.

5. Working with Data Structures

When working with data in programming, it's important to organize and store information efficiently. Python provides several built-in data structures that allow you to group values together, manage collections, and access or manipulate them in powerful ways. The most common are: lists, tuples, dictionaries, and sets.

5.1 Lists

A list is an ordered and mutable collection of items. You can store any data type (numbers, strings, even other lists), change values, and add or remove elements. 

fruits = ["apple", "banana", "cherry"]
    print(fruits[0])          # "apple"
    fruits.append("orange")   # Add to the end
    fruits[1] = "pear"        # Change an item
    fruits.remove("cherry")   # Remove an item
    print(fruits)             # ['apple', 'pear', 'orange']

Lists are very useful when you need to work with a collection of related items — like names, numbers, or inputs from the user.

5.2 Tuples

A tuple is like a list, but immutable — once created, its values cannot be changed. Tuples are useful for fixed collections of values, such as coordinates or days of the week. 

colors = ("red", "green", "blue")
    print(colors[1])  # "green"
    # colors[0] = "yellow"  → Error: tuples cannot be changed

Because they are immutable, tuples are safer to use when you don't want the data to change accidentally.

5.3 Dictionaries

A dictionary stores data in key-value pairs. This is useful when you want to label your data, like storing information about a person. 

person = {
      "name": "Alice",
      "age": 30,
      "city": "London"
    }
    
    print(person["name"])       # Alice
    person["age"] = 31          # Update value
    person["email"] = "a@mail.com"  # Add new key
    del person["city"]          # Remove a key

Dictionaries are extremely flexible and widely used in real applications, especially when managing structured data.

5.4 Sets

A set is an unordered collection of unique items. Sets are useful for removing duplicates and performing mathematical operations like union, intersection, or difference. 

a = {1, 2, 3, 4}
    b = {3, 4, 5, 6}
    
    print(a | b)  # Union: {1, 2, 3, 4, 5, 6}
    print(a & b)  # Intersection: {3, 4}
    print(a - b)  # Difference: {1, 2}

You can convert a list to a set to remove duplicates: 

numbers = [1, 2, 2, 3, 4, 4, 5]
    unique_numbers = set(numbers)
    print(unique_numbers)  # {1, 2, 3, 4, 5}

5.5 Iterating Over Data Structures

You can use for loops to go through lists, tuples, dictionaries, or sets. 

# List
    for fruit in fruits:
        print(fruit)
    
    # Dictionary
    for key, value in person.items():
        print(key + ":", value)

Understanding how to iterate over different structures is essential for processing and analyzing data.

6. Error Handling and Debugging

When writing code, errors are inevitable — especially when you're just starting out. Learning how to handle errors and debug your programs is an essential skill for any programmer. Python provides tools to catch and manage errors gracefully, helping prevent your program from crashing and improving its reliability.

6.1 Common Types of Errors

Before we handle errors, we need to understand the kinds of problems that can occur in a program:

  • SyntaxError: Happens when Python doesn't understand your code. Example: forgetting a colon.
  • NameError: Occurs when a variable or function is used before it is defined.
  • TypeError: Happens when you try to use a value in an incorrect way (e.g., adding a string and an integer).
  • ValueError: Raised when a function receives an argument of the right type but inappropriate value (e.g., converting "abc" to an integer).
  • IndexError: Accessing an element outside the valid range of a list.

6.2 Using try and except

You can use the try and except blocks to "catch" and respond to errors without stopping the whole program. 

try:
        number = int(input("Enter a number: "))
        print("You entered:", number)
    except ValueError:
        print("That's not a valid number!")

In this example, the program will catch the ValueError if the user enters something that's not a number.

6.3 Catching Multiple Exceptions

try:
        num = int(input("Enter a number: "))
        result = 10 / num
    except ValueError:
        print("Please enter a valid number.")
    except ZeroDivisionError:
        print("You can't divide by zero!")

You can handle multiple error types separately to provide more specific feedback to the user.

6.4 The finally Block

You can add a finally block to execute code no matter what happens in the try/except block. 

try:
        f = open("example.txt", "r")
        data = f.read()
    except FileNotFoundError:
        print("File not found.")
    finally:
        print("Execution finished.")

6.5 Debugging Techniques

  • Print statements: Use print() to inspect variable values and flow of the program.
  • Use clear variable names: Makes it easier to read and understand code.
  • Isolate the problem: Comment out sections of code to find where the error happens.
  • Use a debugger: If you're using an IDE like VS Code or Thonny, it includes a debugger to step through your code.
  • Read error messages carefully: Python provides useful messages with the type of error and the line number.

6.6 Raising Custom Errors

You can also create and raise your own errors using raise: 

def check_age(age):
        if age < 0:
            raise ValueError("Age can't be negative")
        else:
            print("Valid age:", age)
    
    check_age(-2)

Tip: Always test your code with both correct and incorrect inputs to make sure it behaves as expected and handles errors gracefully.

7. Working with Files

Many programs need to save data or read information from files. Python makes it easy to work with files using built-in functions. In this section, you'll learn how to read from and write to text files, manage file resources properly, and use different file modes.

7.1 Opening and Reading Files

To open a file for reading, use the open() function with mode "r" (read). Always make sure the file exists before trying to read it. 

file = open("data.txt", "r")
    content = file.read()
    print(content)
    file.close()

You can also read line by line: 

file = open("data.txt", "r")
    for line in file:
        print(line.strip())
    file.close()

7.2 Writing to Files

Use "w" mode to write to a file. If the file already exists, its content will be erased. Use "a" to append content instead. 

file = open("output.txt", "w")
    file.write("This is a line.\n")
    file.write("This is another line.\n")
    file.close()

Appending data: 

file = open("output.txt", "a")
    file.write("New data added at the end.\n")
    file.close()

7.3 Using with Statement

It's recommended to use the with statement when working with files. It automatically closes the file even if an error occurs. 

with open("data.txt", "r") as file:
        lines = file.readlines()
        for line in lines:
            print(line.strip())

7.4 File Modes

  • "r": Read (default). Error if file doesn't exist.
  • "w": Write. Creates new file or overwrites existing one.
  • "a": Append. Adds content at the end of the file.
  • "x": Create. Error if file already exists.
  • "b": Binary mode (e.g., "rb", "wb").

7.5 Writing Lists or Multiple Lines

You can write multiple lines at once using writelines():

lines = ["Line 1\n", "Line 2\n", "Line 3\n"]
    with open("output.txt", "w") as file:
        file.writelines(lines)

7.6 Checking if File Exists

Use the os or pathlib module to check if a file exists before opening: 

import os
    
    if os.path.exists("data.txt"):
        with open("data.txt", "r") as f:
            print(f.read())
    else:
        print("File not found.")

8.1 Calculator App

Build a simple calculator that asks the user for two numbers and an operation (+, –, *, /) and displays the result. Make sure to handle division by zero and invalid input. 

# Sample outline
    num1 = float(input("Enter first number: "))
    num2 = float(input("Enter second number: "))
    op = input("Choose operation (+ - * /): ")
    
    if op == "+":
        print(num1 + num2)
    elif op == "-":
        print(num1 - num2)
    elif op == "*":
        print(num1 * num2)
    elif op == "/":
        if num2 != 0:
            print(num1 / num2)
        else:
            print("Error: Division by zero")
    else:
        print("Invalid operation")
