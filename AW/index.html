Web Applications

HTML, CSS, WordPress, cloud storage, LMS, and Python for web automation

Course Objectives

  • Understand the basics of web applications and the client-server model.
  • Create websites using HTML and CSS with semantic, accessible design.
  • Install and customize WordPress to build functional sites.
  • Manage file and data storage in cloud systems and simple databases.
  • Install and manage a learning platform (Moodle).
  • Write small programs in Python to add automation or interactivity to websites.

Course Structure

  1. UD1 – Introduction to Web Applications: Internet basics, client-server, browsers, protocols.
  2. UD2 – Creating Web Content (HTML, CSS & CMS): Semantic HTML, responsive CSS, and WordPress.
  3. UD3 – Data Storage: Cloud platforms (Drive, Nextcloud), and introduction to databases.
  4. UD4 – Learning Management Systems: Installing and using Moodle as a virtual classroom.
  5. UD5 – Intro to Programming: Python basics and automation in web projects.

Assessment

  • ✅ Practical tasks and lab exercises (HTML/CSS, CMS, Python, LMS)
  • ✅ Unit quizzes or short tests
  • ✅ Final mini-project combining two or more course topics

Recommended Tools

  • Visual Studio Code (or any other editor)
  • Chrome/Firefox with DevTools enabled
  • GitHub for hosting static sites (optional)
  • MDN Web Docs, W3Schools, WordPress Docs, Moodle Docs, Python Docs
